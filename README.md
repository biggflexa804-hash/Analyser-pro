# Derivative Trading Analyzer

A comprehensive Streamlit web application for analyzing derivative trading positions, calculating option prices, and managing risk exposure.

## Features

- **Portfolio Management**: Track options and futures positions
- **Option Pricing**: Black-Scholes calculator with Greeks
- **Implied Volatility**: Calculate IV from market prices
- **Risk Analysis**: Portfolio Greeks and risk metrics
- **Scenario Analysis**: P&L under various market conditions
- **Visualizations**: Interactive charts and sensitivity analysis
